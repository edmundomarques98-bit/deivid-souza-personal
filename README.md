# Deivid Souza — Personal Trainer

Site oficial do acompanhamento personalizado de Deivid Souza.

## Páginas

- Início
- Acompanhamento
- Resultados
- Planos e valores
- Contato direto pelo WhatsApp

## Publicação

O site é exportado como conteúdo estático e publicado automaticamente pelo GitHub Actions no GitHub Pages a cada atualização da branch `main`.

## Desenvolvimento

Requisitos: Node.js 22 e npm.

```bash
npm ci
npm run build:pages
```

O resultado estático é gerado na pasta `out`.
